📚 Components Explained
State

Conversation Phase

A State represents a major phase of the conversation. The agent progresses through states sequentially.

idUnique identifier
labelDisplay name in UI
statusCurrent progress
pending active completed
Task

Action to Perform

A Task is a specific action within a state. Each task may collect multiple deliverables.

idUnique identifier
labelWhat to do
requiredMust complete?
pending in_progress completed
Deliverable

Data to Collect

A Deliverable is information the agent extracts from the conversation.

keyVariable name
typestring, number, boolean
examplesExpected values (optional)
valueCollected data
pending partial completed
